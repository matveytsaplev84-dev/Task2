# Task2
задание 2 экзамен

# Возможности

- **Добавление книг** - добавление новых книг с указанием названия, автора, жанра, года издания и описания
-  **Просмотр книг** - просмотр всех книг с возможностью сортировки и фильтрации
-  **Избранное** - добавление книг в избранное для быстрого доступа и их удаления из избранного
-  **Статус прочтения** - отметка книг как прочитанных/непрочитанных
-  **Поиск** - поиск книг по ключевым словам в названии, авторе или описании
-  **Удаление** - удаление книг из библиотеки
-  **Сохранение данных** - автоматическое сохранение в файл и загрузка при запуске

# Требования
